this is a python ai program that reads a sentence from the user and tells if it is positive negative or neutral.
it has two lists of words one for good words and one for bad words
when you enter a sentence the program changes it to lowercase and checks how many good or bad words are inside
if there are more good words it says positive if there are more bad words it says negative if they are equal it says neutral
it keeps running until you type exit to stop the program
